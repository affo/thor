# Thor

Thor is an actor-driven imperative language.  
This is a port and further development of repository https://github.com/faustoschiavoni/yalang.

## Requirements

Requires Python3 and Java.

```console
make pre
make generate
```
## Run

Run REPL:

```console
export PYTHONPATH=.

./bin/repl
```

Run an example:

```console
./bin/repl ./examples/simple.yl
```
